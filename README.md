# Сайт-визитка — Никита Милашевич

Мобильная одностраничная визитка: производство ВСПОМНИ и стартап DoubleMark.

## Запуск локально

Откройте `index.html` в браузере или запустите простой сервер:

```bash
npx serve .
```

## Контакты

В `config.js` заданы Telegram и телефон:

```js
const TELEGRAM_USERNAME = "Basifilder";
const PHONE_DISPLAY = "8 952 245-30-31";
const PHONE_TEL = "+79522453031";
```

- Кнопка **«Написать в Telegram»** откроет диалог с вами.
- **«Оставить заявку на бету»** откроет форму; после отправки текст заявки уйдёт в Telegram (удобно принимать заявки без сервера).

## Деплой

Загрузите папку целиком на любой хостинг статики (GitHub Pages, Netlify, Vercel, Timeweb и т.д.).
