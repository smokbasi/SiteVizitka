Обо мне

Четыре с половиной года руковожу производством — косметика, продукция для полости рта, парфюмерия. Видел изнутри, сколько решений принимается в Excel, а сколько времени уходит на одни и те же операции в ERP.

Три года назад начал писать код — сначала для своих задач на производстве, потом для заказчиков. Делаю веб-приложения, скрипты автоматизации и интеграции с учётными системами. Беру заказы на разработку.

Ниже — проекты, которые выросли из конкретных проблем: потерянные коды маркировки, ручное заполнение актов, планирование закупок, аналитика выпуска и премий. Каждый решает одну боль, а не «ещё один сервис».

Внутренний инструмент

DoubleMark

DoubleMark — сервис для дублирования кодов маркировки «Честный Знак». Проблема знакома любому производителю: код повредился на линии или потерялся в партии — восстановление превращается в отдельную операцию с риском ошибок.

Сейчас проект на стадии бета-тестирования. Ищу производителей, готовых протестировать и дать обратную связь.

Автоматизация учётных документов

Набор скриптов для документооборота между производством и складским учётом: заполнение акта приёма-передачи МХ-1 из УПД, подстановка кодов номенклатуры МойСклад, вставка КИЗ из Excel в XML-файлы отгрузки.

Задача, которую раньше делали вручную по каждой партии, теперь выполняется за минуты.

Планирование закупок

Интеграция с МойСклад: подтягивает позиции из заказов поставщику, сопоставляет с номенклатурой и формирует таблицу материалов с актуальными ценами и расчётом потребности. Используется для планирования закупок упаковки и комплектующих.

Каталоги сырья

Исследовательские инструменты для закупки парфюмерного сырья: каталоги безспиртовых баз для body mist, сравнительные таблицы силиконовых и углеводородных носителей, инструкция по самостоятельному поиску поставщиков. Данные верифицируются по INCI и сертификатам.
